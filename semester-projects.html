Semester Projects

Hands-on hardware design and electronics projects completed during undergraduate coursework (2022-2023)

Line Following Robot

This project demonstrates the implementation of a line-following robot using purely analog electronics without any microcontrollers. The system utilizes 3 IR sensors strategically positioned to detect the line path and employs only flip-flops and logic gates to process sensor inputs and control the robot's movement in real-time.

Pulse Rate Monitoring System

A wearable pulse rate monitoring system designed using piezoelectric sensors to accurately measure human heart rate. The project focuses on signal processing and amplification techniques to convert mechanical vibrations from pulse beats into measurable electrical signals for real-time heart rate monitoring.
