Slum Detection using U-Net

Binary segmentation of slum areas in high-resolution satellite imagery using deep learning

Project Overview

This project applies cutting-edge deep learning techniques to detect and segment slum regions from high-resolution satellite imagery. Using a sophisticated U-Net architecture with a ResNet-34 encoder, the model was trained on the Mumbai dataset to achieve exceptional segmentation performance.

The results demonstrate the significant potential of AI-powered solutions for urban planning, socio-economic analysis, and policy-making in developing urban areas. This technology can help governments and NGOs better understand urban poverty distribution and plan targeted interventions.

Methodology & Implementation

The project follows a comprehensive deep learning pipeline designed for optimal performance:

  • Data Preprocessing: Carefully preprocessed high-resolution satellite images and corresponding binary masks to ensure optimal model training conditions
  • Architecture Design: Implemented U-Net with ResNet-34 encoder backbone, leveraging transfer learning from pre-trained ImageNet weights for enhanced feature extraction
  • Loss Function Optimization: Utilized a sophisticated combination of Binary Cross-Entropy (BCE) + Dice + Focal Loss to effectively handle class imbalance challenges
  • Training Strategy: Optimized using Adam optimizer with cosine learning rate scheduling for stable convergence and optimal performance
  • Validation: Comprehensive evaluation using multiple metrics including IoU, F1-score, AUC-ROC, and AUC-PR to ensure robust model assessment
