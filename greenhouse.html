Project Overview

The Smart Greenhouse System addresses the growing need for efficient agricultural practices in an era of climate change and food security concerns. This IoT-based solution combines embedded systems, machine learning, and real-time monitoring to create an intelligent farming environment that optimizes plant growth while minimizing resource consumption.

Traditional farming methods often rely on manual monitoring and guesswork, leading to inefficient resource usage and suboptimal crop yields. Our system leverages automation and data-driven insights to revolutionize greenhouse management.

  • Automate environmental monitoring and control systems for optimal plant growth conditions
  • Implement intelligent irrigation systems that reduce water consumption
  • Integrate AI-powered disease detection and agricultural assistance through advanced ML models

System Architecture

System Architecture Diagram

Arduino UNO → Sensors → Raspberry Pi → Web Dashboard

The system architecture follows a hierarchical approach where Arduino UNO handles sensor data collection and actuator control, while Raspberry Pi manages data processing, web services, and external API communications.

DHT11 Sensor

Temperature and humidity monitoring with ±2°C accuracy

Soil Moisture

Capacitive soil moisture sensor for irrigation control

LDR Sensor

Light intensity measurement for optimal lighting conditions

Rain Sensor

Precipitation detection for automatic irrigation adjustment

Automated Control Algorithm

START
├── Initialize Sensors & Actuators
├── Read Environmental Data
│   ├── Temperature & Humidity (DHT11)
│   ├── Soil Moisture Level
│   ├── Light Intensity (LDR)
│   └── Rain Detection
├── Process Data & Make Decisions
│   ├── IF soil_moisture < 30%
│   │   └── Activate Irrigation (5 minutes)
│   ├── IF temperature > 35°C
│   │   └── Activate Cooling Fan
│   ├── IF light_intensity < 300 lux
│   │   └── Activate LED Grow Lights
│   └── IF rain_detected == true
│       └── Disable Irrigation System
├── Send Data to Raspberry Pi
├── Update Web Dashboard
├── Wait 30 seconds
└── REPEAT
